Este repositorio está pensado para aprender a usar Dash usando los datos del paquete de CRAN [nycflights13](https://cran.r-project.org/web/packages/nycflights13/index.html).

Guía para correr el código:
1. Instalar una versión de Python >= 3.8 y jupyter notebook. Puede ser desde anaconda.com.
2. Crear un entorno virtual con el comando ```python -m venv <entorno>```.
3. Ir a la carpeta Intro_Dash y correr el comando ```pip install -r requirements.txt``` para instalar las librerías necesarias.
4. Escribiendo ```jupyter notebook``` en la terminal se va a abrir una ventana en el navegador y ahí hay que seleccionar el archivo Intro_Dash.ipynb.
